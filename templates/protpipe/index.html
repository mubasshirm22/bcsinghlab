Accepts raw amino acid string, single FASTA record, or a UniProt/NCBI accession (e.g. P12345 or NP_000005.2).

2 — Sequence or Accession
3 — Analyses

Basic sequence properties (length, MW, pI, GRAVY, instability) are always computed — no checkbox needed.

Note on Beta modules HMMER (Pfam) and Phobius are connected to EBI web services with medium-high confidence. If either endpoint returns an error, the results page will display a clear diagnostic message. BLAST is fully confirmed and always runs.
Accepted Inputs
  • Raw amino acid string (single-letter codes)
  • Single FASTA record (with or without header)
  • UniProt accession (e.g. P04637)
  • NCBI protein accession (e.g. NP_000535.3)
  • Uploaded .fasta file
What You Get
  • Sequence properties (MW, pI, GRAVY, stability)
  • Top 10 BLAST hits vs SwissProt
  • Pfam domain hits with coordinates
  • Signal peptide detection and cleavage site
  • Transmembrane helix count and topology
  • SVG domain architecture figure
  • Downloadable JSON / TXT / SVG report
Typical Runtime
  • Sequence retrieval: <5 seconds
  • Sequence properties: instant
  • BLAST: 2–10 minutes (SwissProt)
  • Pfam HMMER: ~30–60 seconds
  • Phobius: ~30 seconds
  • BLAST, HMMER, and Phobius run in parallel
