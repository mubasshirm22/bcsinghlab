Accepted Inputs
  • Raw amino acid string (single-letter codes)
  • Single FASTA record (with or without header)
  • UniProt accession (e.g. P04637)
  • NCBI protein accession (e.g. NP_000535.3, XP_, WP_)
  • NCBI mRNA / transcript (e.g. NM_005228.5, XM_) → auto-resolved
  • WGS contig (e.g. GAJC01020720.1) → auto-resolved
  • NCBI gene ID (numeric) → auto-resolved
  • Uploaded .fasta file
What You Get
  • Sequence properties (MW, pI, GRAVY, stability)
  • Top 10 BLAST hits vs SwissProt
  • Pfam domain hits with coordinates
  • Signal peptide detection and cleavage site
  • Transmembrane helix count and topology
  • SVG domain architecture figure
  • Downloadable JSON / TXT / SVG report
Typical Runtime
  • Sequence retrieval: <5 seconds
  • Sequence properties: instant
  • BLAST: 2–10 minutes (SwissProt)
  • Pfam HMMER: ~30–60 seconds
  • Phobius: ~30 seconds
  • BLAST, HMMER, and Phobius run in parallel
